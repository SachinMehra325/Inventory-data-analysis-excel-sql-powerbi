# 📦 Inventory Data Analysis Project

## 📊 Overview
This project focuses on analyzing inventory data using SQL and Power BI to generate business insights and support decision-making.

---

## 🛠️ Tools Used
- SQL (PostgreSQL / pgAdmin 4)
- Power BI
- Excel
- CSV

---

## 📂 Dataset
Inventory dataset containing:
- Product ID
- Product Name
- Stock
- Reorder Level
- Supplier
- Price
- Stock_Status
- Inventory_Value
- Profit_Margin

---

## 🧹 Data Cleaning (SQL)
- Removed duplicate records
- Handled NULL values
- Standardized text format
- Created Stock Status,Inventory_Value,Profit_MarginProfit_Margin column
- Calculated Inventory Value and Profit Margin'
- Analysis All The Business Problem as for Given

---

## 📊 Business Questions Solved

1. Total stock available
2. Total inventory value
3. Low stock items count
4. Pending (out-of-stock) items
5. Stock availability percentage
6. Top 5 products by value
7. Supplier contribution
8. Average product price
9. Highest value product
10. Total stock by product
11. Inventory value by product
12. Stock analysis (product + supplier)
13. Supplier inventory value
14. Profit margin by product
15. Profit margin by supplier

---

## 📈 Dashboard
Built an interactive dashboard in Power BI including:
- KPI Cards (Stock, Value, Low Stock, Pending)
- Top Products Analysis
- Supplier Analysis
- Stock Status Visualization
